# Ventilator-Associated Pneumonia Microbiome Analysis

## Project Overview
This project analyzes 16S rRNA microbiome data and clinical biomarkers to identify microbial signatures associated with ventilator-associated pneumonia (VAP).

## Key Features
- OTU-based microbiome diversity analysis
- Alpha and beta diversity metrics
- Biomarker correlation analysis (IL-6, TNF-α, IL-1β)
- Logistic regression and LASSO modeling
- ROC-AUC evaluation
- Microbiome clustering
- Clinical integration (RWE approach)

## Tools Used
- Python (pandas, scipy, sklearn)
- Statistical modeling
- Microbiome analytics

## Results
- Reduced diversity observed in VAP patients
- Strong association between inflammatory biomarkers and dysbiosis
- Predictive model achieved ROC-AUC of ~0.XX

- notebooks/: analysis workflow
- outputs/: results and plots
